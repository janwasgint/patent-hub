# Patent Hub
```
cd ethereum-private-network/truffle-code

npm (or yarn) install

npm (or yarn) install bootstrap

ganache-cli -d -m "raven shock define wish brown cloth twin win weasel stable tone share"

ipfs daemon

truffle compile
truffle migrate

Find the PatentHub deployment address in the terminal output and copy it to src/const.

npm (or yarn) start

In Metamask -> Settings -> Security & Privacy -> untoggle privacy mode
```
